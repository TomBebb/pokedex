# Pokedex Vue frontend

[View on Vercel](https://tomb-pokedex.vercel.app/)

This is a basic Pokedex frontend using the public [PokeAPI API](https://pokeapi.co/).
This uses the following components in its stack:

- Prettier: a CLI for formatting code using a config
- Vue
- Vite as frontend dev/build tool
- TailwindCSS: a CSS micro-framework
